# Event Ticket Booking System

## Project Overview
A full-stack event ticket booking platform where users can browse events, select seats, book tickets, and manage bookings.

## Features
- User Registration & Login
- JWT Authentication
- Event Management
- Seat Selection
- Ticket Booking
- Redis Seat Locking
- Admin Dashboard
- Docker Deployment
- AWS Hosting

## Tech Stack
### Backend
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- Redis

### Frontend
- React
- TypeScript
- Tailwind CSS

### DevOps
- Docker
- AWS
